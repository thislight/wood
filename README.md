# wood
The faster way to build web application by tornado.

## Quick Start
wood is alpha now, you only can install wth code.

1.Install `git`.
with apt:
````
apt install git
````
with dnf:
````
dnf install git
````

2.Clone & Install wood
````
cd ~
git clone https://github.com/thislight/wood.git
cd wood
python setup.py
````
